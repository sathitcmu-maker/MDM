SET Board Requirements

กฎ SET ที่ Board ต้องทำตาม

SET CG Code
+

SET CG Code กำหนดว่า Board ต้องมีองค์ประกอบและกระบวนการที่ชัดเจน — ถ้า Board composition ไม่ถูกต้อง SEC อาจไม่อนุมัติ filing

Board size

ไม่น้อยกว่า 5 คน — LACO ควรมี 7 คน (1 Chairman + 1 CEO + 3 Proterra nominees + 2 Independent) เพื่อ balance ระหว่าง PE nominee และ Independent

Independent Director ≥ 1/3

ถ้า Board มี 7 คน → ต้องมี Independent อย่างน้อย 3 คน — Proterra nominees ไม่นับเป็น Independent เพราะมี material relationship กับ major shareholder

Chairman ≠ CEO (SET requirement)

ต้องแยก Chairman และ CEO ออกจากกันชัดเจน — Aphirak เป็น CEO ดังนั้น Chairman ต้องเป็นคนอื่น และควรเป็น Independent Director

Board Charter

ต้องมี Board Charter ที่กำหนด role, responsibility, authority, meeting frequency, และ evaluation process — document ที่ต้อง disclose บน SET website หลัง listing

Board evaluation

ทำ board performance evaluation ปีละ 1 ครั้ง (whole board + individual + committee) — ผลการประเมินต้องรายงานใน Annual Report

Independent ≥ 3/7Chairman ≠ CEOBoard Charter required

การจัดการ PE nominee ใน Board

Proterra Asia
+

PE nominee rights

Proterra Asia (PAFF3) มี Tai Lin, Koh-Peng Chua, William So เป็น nominees — สิทธิ์ nominate จะลดลงตามสัดส่วนถือหุ้นหลัง IPO เมื่อมี public float เพิ่มขึ้น

Exit timeline awareness

PE fund มี investment horizon 5–7 ปี — PAFF3 ลงทุน LACO ปี 2025 ดังนั้น exit window คาดว่าประมาณ 2030–2032 Board ต้องรับรู้ timeline นี้ในการวาง long-term strategy

Board independence post-IPO

หลัง IPO สัดส่วน Proterra ลดลง → Independent Directors มี voice มากขึ้น → governance balance ดีขึ้น — ออกแบบ Independent Director selection process ให้ดีตั้งแต่ต้น

LP reporting obligation

Proterra Asia ต้องรายงาน ESG, financial performance, governance ให้ LP (Limited Partners) ของ fund — Board ต้องเข้าใจว่า reporting requirement ของ PE fund สูงกว่า SET minimum

Director Qualifications

คุณสมบัติที่ SEC กำหนด

SEC Requirement
+

SEC กำหนด negative qualifications สำหรับ Director — ถ้ากรรมการมีประวัติที่ขัดต่อเงื่อนไข SEC จะไม่อนุมัติ IPO filing ต้องตรวจสอบก่อน propose รายชื่อ

Negative qualifications (SEC)

ไม่มีประวัติอาชญากรรมทางการเงิน, ไม่เคยถูก SEC สั่งห้ามดำรงตำแหน่ง, ไม่เป็นบุคคลล้มละลาย, ไม่มี conflict of interest ที่ material กับ LACO — ต้องทำ background check ทุกคน

Independent Director definition (SEC)

ถือหุ้นไม่เกิน 1% ของ voting shares, ไม่มีความสัมพันธ์ทางธุรกิจกับ LACO เกิน 3 ปีย้อนหลัง, ไม่เป็น key management ของ LACO หรือ related party ในปัจจุบัน

Board diversity (CG Code)

SET CG Code ส่งเสริม board diversity: gender, age, expertise — LACO ควร consider female director อย่างน้อย 1 คน เพื่อ ESG rating ที่ดีขึ้น (FTSE Russell ให้ weight กับ board diversity)

Background check requiredGender diversity signal

ตรวจสอบก่อน filing: Proterra Asia nominees ทั้ง 3 คน (Tai Lin, Koh-Peng Chua, William So) ต้องผ่าน SEC qualification check — Singapore executives ที่ไม่เคยอยู่ใน Thai listed company อาจต้องเตรียมเอกสารเพิ่มเติม

Audit Committee (AC)

ตรวจสอบงบการเงิน · Internal control · Auditor oversight

SET Mandatory
+

Audit Committee คือ committee ที่ SET/SEC ให้ความสำคัญมากที่สุด — ต้องประกอบด้วย Independent Directors ทั้งหมด และอย่างน้อย 1 คนต้องมี financial expertise

Composition (SET requirement)

≥ 3 Independent Directors ทั้งหมด, อย่างน้อย 1 คน มีความรู้ด้าน accounting/finance — Chairman ของ Audit Committee ต้องไม่ใช่ Chairman of the Board

Core mandate

Review financial statements ก่อน Board approve, oversee internal audit function, select/recommend external auditor ต่อ AGM, review related-party transactions, evaluate adequacy of internal controls (COSO/SEC framework)

Internal Audit oversight

Internal Audit รายงานตรงต่อ Audit Committee (ไม่ใช่ CEO) เพื่อ independence — AC approve Internal Audit plan ทุกปี, review ผลการ audit, และ track remediation ของ audit findings

MDM Manager interface

MDM Manager (Boss) เป็นผู้นำเสนอเอกสารต่อ Audit Committee: internal control adequacy report, risk management update, legal compliance status — เป็น key presenter ไม่ใช่ decision maker

Related-Party Transactions

Conflict of interest · SEC disclosure

SEC Critical
+

Related-Party Transactions (RPT) เป็น red flag ที่ SEC ดูเป็นพิเศษในทุก IPO — ถ้า LACO มี transactions กับ บริษัทที่ผู้ถือหุ้นใหญ่หรือ Board members มีส่วนได้เสีย ต้อง disclose และผ่าน Audit Committee ก่อน

RPT Policy

กำหนด policy ว่า RPT ทุกรายการต้องผ่าน Audit Committee review ก่อน เพื่อยืนยัน arm's length terms — ห้าม CEO หรือกรรมการที่มี conflict of interest เข้าร่วม decision

ทำ RPT audit ก่อน filing — identify ทุก transaction กับ related parties ย้อนหลัง 3 ปี ถ้ามี non-arm's length terms ต้อง remediate ก่อน SEC submission
